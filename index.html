ポケモンの実数値計算

▼Lv.50のポケモンの実数値計算式

ポケモンのステータスを計算するには3値、つまり【種族値】【努力値】【個体値】が必要です。

また、性格補正値も理解しておく必要があります。

●HPの計算式

(種族値+個体値÷2+努力値÷8)+60

●HP以外の能力の計算式

＊性格補正は1.1倍と0.9倍とする。

{(種族値+個体値÷2+努力値÷8)+5}×性格補正

▼例　Lv.50のドヒドイデ(ずぶとい)のステータス

/ H A B C D S
種族値 50 63 152 53 142 35
個体値 31
努力値 252 0 252 0 0 0
実数値 157 74 224 73 162 55
このドヒドイデのステータスを求める計算式はそれぞれ以下の通りになる

*性格補正以外で小数点以下が出ると出た時点で切り捨てる

  • ●HP=(50+31÷2+252÷8)+60
  • =(50+15.5+31.5)+60
  • =97+60
  • =157
  • ●A={(63+31÷2+0÷8)+5}×0.9
  • ={(63+15)+5}×0.9
  • =83×0.9
  • =74
  • ●B={(152+31÷2+252÷8)+5}×1.1
  • ={(152+15.5+31.5)+5}×1.1
  • =224
  • ●C={(53+31÷2+0÷8)+5}×1
  • ={(53+15)+5}
  • =73
  • ●D={(142+31÷2+0÷8)+5}×1
  • ={(142+15)+5}
  • =162
  • ●S={(35+31÷2+0÷8)+5}×1
  • ={(35+15)+5}
  • =55

▼Lv.Xのポケモンの実数値計算式

X＝レベルの数値とする

●HPの計算式

(種族値×2+個体値+努力値÷4)×(0.01X)+(10+X)

●Hp以外の能力の計算式

{(種族値×2+個体値+努力値÷4)×(0.01X)+5}×性格補正
